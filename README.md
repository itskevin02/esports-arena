EA Sports Arena Manager, es una plataforma backen desarrollada con arquitectura de Microservicios utilizando Spring Boot.
El Objetivo del proyecto es administrar informacion relacionada en el entorno competitivo de los usuarios, equipos, torneos y participar mediantes APIs Rest.

Actualmente el Proyecto se Encuentra en Desarrollo academico utilizando Java, Maven y spring boot.
